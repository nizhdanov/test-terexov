# Тестовое в Texerov

## Задание

> UX и a11y
> SPA-приложение со страницей входа по email и паролю, но без использования UI-библиотек.
> Продумать отправку email и пароля на сервер — достаточно замокать `fetch()`.

Установка зависимостей

```bash
npm i
```

Запуск dev сервера

```bash
npm run dev
```
